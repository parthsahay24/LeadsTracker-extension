Install Leads Tracker

Follow these simple steps to add the extension to Chrome.

  1. 1 Download the extension ZIP file and extract (unzip) the downloaded folder.

    2. Steps to install:


  2. 1 Open chrome://extensions in Chrome.
  3. 2 Enable Developer mode (top-right).
  4. 3 Click Load unpacked and select the extracted folder.
  5. 4 LeadsTracker extension added to browser successfully.
